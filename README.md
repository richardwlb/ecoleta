## About Ecoleta

Ecoleta is an App Mobile and Web used to people to find places to discard recycle trash. It was build during the "Next Level Week" by Rocketseat.

## Technologies

- Node
- TypeScript
- React
- React native
- Expo

#### How to use

```sh
  $ git clone https://github.com/richardwlb/ecoleta.git
```

```sh
  # Dependencies
  $ npm install

  ## Database creation
  $ cd server
  $ npm run knex:migrate
  $ npm run knex:seed

  # API start
  $ npm run dev

  # WEB Start
  $ cd web
  $ npm start

  # Mobile Start
  $ cd mobile
  $ npm start
```
